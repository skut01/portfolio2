Application d'agenda partagé

Informations

  • Date du projet: été 2023
  • Languages: Python

Réalisation d'une application d'agenda partagé pour l'entreprise BOSSART comptant 8 concessions automobiles. L'objectif était de pouvoir importer des données depuis le site SILAE RH puis de les sélectionner et les afficher sur l'application.

    Compétences acquises:

  • Analyse des besoins du client
  • Développement en entreprise
  • Automatisation

    Méthodologie:

  • Réunion avec le client
  • Développement des tâches précisées par le client
  • Démo
  • Réitération
