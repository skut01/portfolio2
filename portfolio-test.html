Framework de test

Informations

  • Date du projet: BUT semestre 3, 2023
  • Languages: C

Développement d'un framework permettant de lancer des séries de test en C.

    Compétences acquises:

  • développement en C
  • Framework
  • Forks
  • Threads

    Méthodologie:

  • Compréhension du code éxistant
  • Développement des modes de tests principaux (avec forks, sans forks, forks en parallèles)
  • Implémentation des options (logs, timeout, mode silencieux)
